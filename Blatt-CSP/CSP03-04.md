# CSP.04
**Gegeben:** v1 = 2

## (1) Kantenkonsistenz (volle Propagation)
- Aus v1 + v2 = 3 → **v2 = 1**
- Aus v1 ≤ v3 und v2 + v3 ≤ 3 → **v3 = 2**
- Aus v3 ≠ v4 → 2 entfernen → **v4 = {0,1,3,4,5}**

### Enddomains (Kantenkonsistenz)
- v1 = {2}
- v2 = {1}
- v3 = {2}
- v4 = {0,1,3,4,5}

## (2) Forward Checking (nur direkte Nachbarn von v1)
- v2: 2 + v2 = 3 → **v2 = {1}**
- v3: 2 ≤ v3 → **v3 = {2,3,4,5}**
- v4: unverändert → **{0,1,2,3,4,5}**

### Enddomains (Forward Checking)
- v1 = {2}
- v2 = {1}
- v3 = {2,3,4,5}
- v4 = {0,1,2,3,4,5}

## Vergleich
- **Kantenkonsistenz** schränkt stärker ein (v3 und v4 reduziert).
- **Forward Checking** schränkt nur direkte Nachbarn von v1 ein.
